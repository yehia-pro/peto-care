# 🏥 نظام تشغيل موقع العيادة البيطرية

## 📋 نظرة عامة
هذا النظام يتيح لك تشغيل موقع العيادة البيطرية بنقرة واحدة فقط دون الحاجة لكتابة الأوامر أو التنقل بين المجلدات.

## 🚀 الخيارات المتاحة

### 1. **السكريبت الأساسي (الأنسب للمبتدئين)**
```batch
start-website.bat
```
✅ يفتح الموقع فقط مع واجهة أمامية
✅ أسهل وأسرع طريقة
✅ مناسب للاستخدام اليومي

### 2. **السكريبت المتقدم (للمطورين)**
```batch
start-full-website.bat
```
✅ يفتح الخادم الخلفي والأمامي في نوافذ منفصلة
✅ مناسب للتطوير والتعديلات
✅ يعرض جميع سجلات النظام

### 3. **السكريبت الذكي (PowerShell)**
```powershell
.\start-website.ps1
```
✅ يحتوي على خيارات متقدمة
✅ يمكن التحكم فيه من سطر الأوامر
✅ يدعم إعادة التشغيل والفحص

### 4. **الأمر العام (من أي مكان)**
```batch
start_site.bat
```
✅ يمكن حفظه في أي مكان على الكمبيوتر
✅ يعمل من أي مسار في نظام الملفات
✅ لا يحتاج للانتقال إلى مجلد المشروع

## 📦 الإعداد السريع

### الخطوة 1: تشغيل إعداد النظام
انقر نقراً مزدوجاً على:
```batch
setup-website.bat
```

### الخطوة 2: استخدام الاختصار
سيتم إنشاء اختصار على سطح المكتب باسم:
**"موقع العيادة البيطرية"**

### الخطوة 3: التشغيل اليومي
انقر نقراً مزدوجاً على الاختصار للبدء!

## 🎯 الخصائص والمميزات

### ✅ فحص تلقائي
- يتحقق من تثبيت Node.js تلقائياً
- يتحقق من وجود ملفات المشروع
- يثبت التبعيات المفقودة تلقائياً

### ✅ واجهة مستخدم واضحة
- رسائل باللغة العربية الكاملة
- أيقونات واضحة لكل حالة
- ألوان مميزة للنجاح والخطأ

### ✅ معالجة الأخطاء الذكية
- يعرض رسائل خطأ واضحة
- يقترح حلولاً للمشاكل
- يتوقف بأمان عند حدوث خطأ

### ✅ فتح تلقائي للمتصفح
- يفتح المتصفح تلقائياً
- يتوجه مباشرة إلى الصفحة الرئيسية
- ينتظر حتى يبدأ الخادم تماماً

## 🔧 الأوامر المتقدمة (PowerShell)

```powershell
# عرض المساعدة
.\start-website.ps1 -Help

# فحص حالة الموقع
.\start-website.ps1 -Check

# إيقاف الموقع
.\start-website.ps1 -Stop

# إعادة التشغيل
.\start-website.ps1 -Restart

# تثبيت اختصار جديد
.\start-website.ps1 -Install
```

## 📁 ملفات النظام

| الملف | الوظيفة |
|-------|---------|
| `start-website.bat` | السكريبت الأساسي للمبتدئين |
| `start-full-website.bat` | السكريبت المتقدم للمطورين |
| `start-website.ps1` | سكريبت PowerShell الذكي |
| `start_site.bat` | الأمر العام من أي مكان |
| `setup-website.bat` | إعداد النظام والاختصارات |
| `quick-commands.bat` | أوامر سريعة متعددة |

## 🌐 روابط الموقع

- **الرئيسية:** http://localhost:5173/
- **التسجيل:** http://localhost:5173/register
- **تسجيل الدخول:** http://localhost:5173/login

## 💡 نصائح الاستخدام

### للمبتدئين:
1. استخدم `start-website.bat` فقط
2. انتظر حتى يفتح المتصفح تلقائياً
3. لا تغلق نافذة CMD حتى تنتهي

### للمطورين:
1. استخدم `start-full-website.bat` للحصول على جميع السجلات
2. استخدم PowerShell للمزيد من الخيارات
3. تحقق من حالة الموقع باستخدام `-Check`

### للاستخدام اليومي:
1. شغل `setup-website.bat` مرة واحدة فقط
2. استخدم الاختصار على سطح المكتب
3. استمتع بالموقع بنقرة واحدة!

## 🆘 استكشاف الأخطاء

### المشكلة: لا يفتح الموقع
**الحل:** تأكد من أنك لا تستخدم المنفذ 5173 في تطبيق آخر

### المشكلة: خطأ في Node.js
**الحل:** ثبت Node.js من https://nodejs.org/

### المشكلة: ملفات المفقودة
**الحل:** تأكد من تشغيل السكريبت من مجلد المشروع الصحيح

## 📞 الدعم

إذا واجهت أي مشاكل:
1. تأكد من أن جميع الملفات موجودة
2. تحقق من صلاحيات المسؤول إذا لزم الأمر
3. استخدم `setup-website.bat` لإعادة الإعداد
